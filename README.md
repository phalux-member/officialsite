# PHALUX Website

PHALUXの公式ウェブサイト用リポジトリです。

---

## 💻 技術情報

- **GitHub Pages** で静的サイトとしてホスティング  
- **HTML / CSS / JavaScript (jQuery)** を使用  
- **フォーム送信**: Formspree を利用  
- **SNS埋め込み**: JavaScript による Twitter タイムライン表示  
- **独自ドメイン**: https://phalux.com

---
